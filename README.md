# Qt5Application

Qt5Application is a simulated video player desktop application developed with **Qt 5.12.1**, **libvlcplayer**, and **Qt Widgets**.  
The project is intended to demonstrate multimedia interface design and video playback simulation in a Qt-based desktop environment.

## Features

- Simulated video player interface
- Video playback integration based on libvlcplayer
- Desktop GUI built with Qt Widgets
- Basic multimedia interaction controls
- Practice-oriented application structure for Qt development

## Technology Stack

- **C**
- **C++**
- **Qt 5.12.1**
- **libvlcplayer**
- **Qt Widgets**
- **NSIS** for installer-related configuration

## Project Purpose

This project was built as a practice and demonstration application for integrating video playback capabilities into a Qt desktop program. It explores GUI design, multimedia control logic, and external library integration.

## Getting Started

### Requirements

- Qt 5.12.1
- libvlcplayer
- A compatible C/C++ build environment
- Qt Creator or another supported build tool

### Build and Run

1. Clone the repository
2. Make sure Qt 5.12.1 and libvlcplayer are installed correctly
3. Open the project in Qt Creator
4. Configure the build environment
5. Build and run the application

## Usage

After starting the application, users can interact with the simulated video player interface and test the playback-related features provided by the project.
